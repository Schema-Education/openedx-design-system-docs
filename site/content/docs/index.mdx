# Welcome to the Open edX Design System

This site is the canonical documentation and component registry for the Open edX Design System.
It connects **Paragon UI components**, **Figma designs**, and **live code** in a single, browsable source of truth spanning every Open edX micro-frontend (MFE).

## What this site will become

When Phase 2a ships, this hub will provide:

- A **browsable component registry** crawled from all Open edX MFE source trees
- **Atomic design taxonomy** — every component classified as atom, molecule, organism, template, or page
- **Figma Code Connect integration** — Figma node IDs linked to live code for each registered component
- **Multi-MFE consumer map** — know which MFEs consume a given component before you change it
- **Accessibility status** — AA/AAA conformance per component
